 ## Getting Started

Clone from Github 

1. Clone this repo

    ```sh
    git clone https://github.com/olamilekan21/playon.git
    ```

2. Go into the project root directory

    ```sh
    cd playon
    ```

3. Install JS dependencies

    ```sh
    yarn
    ```

4. Start the dev server

    ```sh
    yarn dev
    ```
